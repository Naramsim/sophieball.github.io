关于我们

Welcome home, girls developers

Code Girls“码妞儿”是一个全国性的非营利组织，我们提供简单易学的计算机科学教育，
让女孩们成为明天IT的技术人员所需要的软硬技能。我们不只是提供接触技术，我们培养未来的工程师。

2015夏令营

女生为什么能学好编程

编程的门，一直都是虚掩盖着的

    • 球球

      我向世界解释一万次女生也会喜欢编程，也很适合编程，都比不上这批学员在这7天里写出来的9款精美小游戏有说服力。从零基础到小游戏，这次7天的体验营也许被我设计成了地狱模式，但大家都接受了挑战，坚持到了最后，惊艳了大家。成为一名程序员的路漫长艰辛，消除性别成见的路也许更加曲折难行，今天迈出了第一步，今后还需跋涉千千万万里。我们不是女汉子，我们只是更优秀。

    • 李教授

      希望她们能发现编程的乐趣和神奇之处，将来能为了自己的乐趣编程，因为编程也是一种自由的艺术。

    • Grendal

      看到你们最后做出来的一个个高大上的虐心的狂拽炫酷屌炸天的游戏我真的觉得这真是极好的！ 不得不赞一个说你们真的很聪明可以很快地get到我的BST遍历很快地get到我讲的开发思路并且还很快地get到我的冷笑话！！！话说回来，这个训练营很珍贵很短暂，你们除了学到很多奇奇怪怪的CS的知识以及听懂了很多奇奇怪怪的CS的笑话之外，我觉得最重要的一点就是你们在发现自己的另外一个小宇宙！CS可以是很有趣的很刺激的很酷炫的而且最最最主要的就是CS更是很reachable feasible possible and affordable的！ 真的很开心认识大家可以跟大家分享CS的美妙之处！很可惜由于工作原因我能呆的时间不是很长，不过我也从这短短的几天里面我也向你们学到了很多东西！谢谢！最后，坏坏地祝愿你们在CS的水潭里面越走越深～以后欢迎跟我讨论各种烧脑的数学题计算机问题或者脑经急转弯～

    • 宛琦姐姐

      从零基础开始，学三天写三天，就能做出自己的小游戏，真的是非常了不起大成就！希望这一批女生能在这七天里感受到编程的魅力，同时也认识到自己的潜力。更重要的是，希望她们在日后的生活中都能更加勇敢地去尝试，让生活的精彩程度呈指数形式增长。

    • Windson

      要对学习编程的女生的颜值抱有期望，更要对学习编程女生的能力抱有期望。
